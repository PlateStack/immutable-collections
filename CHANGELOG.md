## [0.1.0-alpha] - 2017-05-26
### Implements
- Immutable API for:
  - Map (linked)
  - HashMap
  - SortedMap
  - NavigableMap
  - Collection
  - List
  - Set (linked)
  - HashSet
  - SortedSet
  - NavigableSet
